# 🚨 FRED 宏观金融危机监察系统

## 📋 系统概述

基于你提供的专业框架和我们收集的FRED数据，我们成功构建了一个完整的宏观金融危机监察系统。该系统能够：

- **实时监控** 29个关键宏观经济指标
- **历史对比** 与26个历史危机期间进行基准比较
- **风险评分** 为每个指标提供0-100分的风险评分
- **智能报告** 生成中文Markdown报告，便于普通人理解

## 🎯 核心功能

### 1. 指标监控
- **收益率曲线**: T10Y3M, T10Y2Y (倒挂检测)
- **信用利差**: BAMLH0A0HYM2, BAA10YM (风险溢价)
- **流动性指标**: SOFR, TEDRATE, CPN3M (资金压力)
- **压力指数**: STLFSI3, NFCI, VIXCLS (综合压力)
- **宏观指标**: GDP, INDPRO, UMCSENT, HOUST (基本面)
- **信贷指标**: TDSP, TOTLL, TOTALSA (债务水平)

### 2. 危机基准
系统基于26个历史危机期间计算基准值：
- 大萧条 (1929-1933)
- 全球金融危机 (2007-2009)
- 新冠疫情 (2020)
- 地区银行危机 (2023)
- 等更多历史危机期间

### 3. 风险评分机制
- 🔴 **高风险** (80-100分): 接近历史危机水平
- 🟡 **中风险** (60-79分): 高于历史正常水平
- 🟢 **低风险** (40-59分): 处于历史正常范围
- 🔵 **极低风险** (0-39分): 低于历史正常水平

## 🚀 使用方法

### 运行监控系统
```bash
# 进入项目目录
cd C:\Users\admin\上网

# 运行危机监控
python -m scripts.crisis_monitor
```

### 每日 9:00 自动运行并发邮件
1) 设置环境变量 `SENDGRID_API_KEY`
2) 编辑 `config/email_settings.yaml`（发件人/收件人）
3) 运行：
```
scripts\create_daily_task.bat
```

### 输出文件
系统会在 `outputs/crisis_monitor/` 目录下生成：
- **CSV报告**: `crisis_report_YYYYMMDD_HHMMSS.csv`
- **Markdown报告**: `crisis_report_YYYYMMDD_HHMMSS.md`
- **JSON数据**: `crisis_report_YYYYMMDD_HHMMSS.json`

## 📁 系统架构

### 配置文件
- `config/crisis_indicators.yaml` - 指标配置
- `config/crisis_periods.yaml` - 历史危机期间配置

### 核心脚本
- `scripts/crisis_monitor.py` - 主监控程序
- `scripts/fred_http.py` - FRED API客户端
- `scripts/supplement_crisis_detection_data.py` - 数据补充脚本

### 数据存储
- `data/fred/categories/Crisis_Detection/` - 危机检测相关数据
- `data/fred/categories/` - 按分类存储的FRED数据

## 🔧 技术特点

### 数据处理
- **智能变换**: 支持level, diff, mom_pct, yoy_pct等变换
- **月度标准化**: 自动转换为月末月度数据
- **缺失值处理**: 智能处理数据缺失情况

### 基准计算
- **多危机期间**: 基于26个历史危机期间
- **统计量计算**: 中位数、四分位数、均值、标准差
- **Z分数计算**: 相对于历史均值的标准化分数

### 风险评分
- **方向感知**: 根据指标特性调整风险方向
- **权重调整**: 不同基准类型使用不同权重
- **分数标准化**: 0-100分标准化评分

## 📈 监控指标详解

### 收益率曲线类
- **T10Y3M**: 10年期-3个月国债利差，倒挂预示衰退
- **T10Y2Y**: 10年期-2年期国债利差，另一重要倒挂指标

### 信用利差类
- **BAMLH0A0HYM2**: 高收益债OAS，反映信用风险
- **BAA10YM**: 投资级信用利差，企业信用风险指标

### 流动性类
- **SOFR**: 隔夜融资利率，短期流动性指标
- **TEDRATE**: TED利差，银行间信用风险
- **CPN3M**: 商业票据利率，企业融资成本

### 压力指数类
- **STLFSI3**: 圣路易斯金融压力指数
- **NFCI**: 芝加哥金融状况指数
- **VIXCLS**: 市场波动率指数

### 宏观基本面类
- **GDP**: 国内生产总值增长率
- **INDPRO**: 工业生产指数
- **UMCSENT**: 密歇根消费者信心
- **HOUST**: 新屋开工数量

### 信贷债务类
- **TDSP**: 家庭债务偿付比率
- **TOTLL**: 银行总贷款增长率
- **TOTALSA**: 消费者信贷增长率
