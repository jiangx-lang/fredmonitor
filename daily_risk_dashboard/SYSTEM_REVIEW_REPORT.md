### 3. 计算逻辑检查 ✅

#### 评分算法
```python
# 1. 基础分位数评分
percentile = (historical_data <= current_value).mean()
base_score = percentile * 100

# 2. 方向调整
if direction == 'down_is_risk':
    percentile = 1 - percentile

# 3. 动量加分
momentum_score = max(1日动量, 5日动量) * 5
final_score = min(100, base_score + momentum_score)

# 4. 共振检测
if 同组内≥2个指标>90%分位数:
    bucket_score += 5分

# 5. 加权总分
total_score = Σ(bucket_score × bucket_weight)
```

#### 计算参数
- **历史回看**: 5年 (1260个交易日)
- **动量窗口**: 1日和5日
- **动量加分**: 最多5分
- **共振阈值**: 90%分位数
- **共振加分**: 每组最多5分
