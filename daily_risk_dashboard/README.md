# FRED 高频风险监控系统

## 📋 系统概述

这是一个基于FRED数据的日度高频风险监控系统，专门用于捕捉市场极端风险情绪和牛市逃顶信号。

## 🎯 系统特点

- **高频敏感**: 使用日度/周度数据，比传统月频指标更灵敏
- **多维度监控**: 涵盖波动率、信用、流动性、风险偏好等5大维度
- **智能评分**: 基于历史分位数 + 动量加分 + 共振检测
- **可视化丰富**: 生成专业的风险面板图片
- **即插即用**: 与现有crisis_monitor系统并存，独立运行

## 📊 监控指标

### A. 市场波动与期限结构 (权重30%)
- **VIXCLS**: VIX波动率指数
- **SKEW**: SKEW指数（极端风险偏好）
- **T10Y3M**: 10年期-3个月国债利差
- **T10Y2Y**: 10年期-2年期国债利差
- **MOVE_PROXY**: 美债波动率代理指标

### B. 信用与融资压力 (权重30%)
- **BAMLH0A0HYM2**: 高收益债券风险溢价
- **BAMLC0A0CM**: 投资级债券风险溢价
- **TEDRATE**: TED利差
- **BAA10Y**: BAA级公司债与10年期国债利差

### C. 流动性与资金面 (权重20%)
- **RRPONTSYD**: 隔夜逆回购余额
- **WSHOMCB**: 美联储总资产
- **WTREGEN**: 财政部TGA账户
- **IORB_EFFR_SPRD**: 准备金利率与联邦基金利率利差

### D. 风险偏好与大类资产 (权重10%)
- **DTWEXBGS**: 贸易加权美元指数
- **DCOILWTICO**: WTI原油价格
- **GOLDAMGBD228NLBM**: 黄金价格
- **SPX_UTIL_RATIO**: 标普500与公用事业股比值

### E. 金融压力综合 (权重10%)
- **STLFSI2**: 圣路易斯金融压力指数
- **NFCI**: 芝加哥金融状况指数

## 🎨 评分机制

### 1. 基础评分
- 每个指标计算5年历史分位数
- 根据风险方向调整：上升=风险上升 或 下降=风险上升
- 基础评分 = 分位数 × 100

### 2. 动量加分
- 计算1日和5日变化量的历史分位数
- 动量加分 = max(1日动量, 5日动量) × 5分

### 3. 共振检测
- 同组内≥2个指标同时>90%分位数时
- 该组获得+5分共振加分

### 4. 最终评分
- 最终评分 = min(100, 基础评分 + 动量加分 + 共振加分)
- 总分 = Σ(分组平均分 × 分组权重)

## 🚀 使用方法

### 1. 测试系统
```bash
cd D:\fred_crisis_monitor\daily_risk_dashboard
python test_risk_dashboard.py
```

### 2. 运行监控
```bash
python risk_dashboard.py
```

### 3. 自动任务
在Windows任务计划程序中设置：
- 每天06:35 AM运行 `python risk_dashboard.py`

## 📁 输出文件

### 图片输出
- `outputs/risk_dashboard_YYYYMMDD_HHMMSS.png`
- 包含总体风险评分、风险等级、触发统计、指标详情表格

### 数据输出
- `outputs/risk_dashboard_YYYYMMDD_HHMMSS.json`
- 包含所有指标的详细数据和评分

## 🎯 风险等级

- **🔵 极低风险** (0-35分): 市场情绪稳定
- **🟢 低风险** (35-65分): 正常波动范围
- **🟡 中风险** (65-80分): 需要关注
- **🔴 高风险** (80-100分): 极端风险，建议减仓

## ⚙️ 配置说明

配置文件: `config/risk_dashboard.yaml`

主要配置项:
- `lookback_years`: 历史数据回看年数 (默认5年)
- `buckets`: 风险分组配置
- `scoring`: 评分参数
- `heatmap_thresholds`: 风险等级阈值

## 📈 与crisis_monitor的区别

| 特性 | crisis_monitor | risk_dashboard |
|------|----------------|----------------|
| 数据频率 | 月频/季频 | 日频/周频 |
| 监控重点 | 长期危机趋势 | 短期风险情绪 |
| 指标数量 | 26个宏观指标 | 20个高频指标 |
| 更新频率 | 日/周 | 日 |
| 适用场景 | 危机预警 | 牛市逃顶 |

## ⚠️ 注意事项

1. **数据依赖**: 需要稳定的FRED API访问
2. **缓存机制**: 自动缓存数据，避免重复下载
3. **容错处理**: 单个指标失败不影响整体运行
4. **仅供参考**: 不构成投资建议，请结合其他分析
