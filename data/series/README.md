# FRED 数据统一数据库

## 📋 概述

本目录包含所有FRED数据的统一存储，为crisis_monitor主程序提供标准化的数据访问接口。

## 📁 数据目录结构

```
data/series/
├── README.md                    # 本文件
├── data_summary.json           # 数据摘要信息
├── {SERIES_ID}.csv             # 原始数据文件
├── {SERIES_ID}_YOY.csv         # YoY数据文件
└── {SYNTHETIC_ID}.csv          # 合成指标文件
```

## 📊 数据文件格式

### 原始数据文件 (`{SERIES_ID}.csv`)
```csv
date,value
2020-01-01,100.5
2020-02-01,101.2
...
```

### YoY数据文件 (`{SERIES_ID}_YOY.csv`)
```csv
date,yoy_pct,original_value
2020-01-01,2.5,100.5
2020-02-01,1.8,101.2
...
```

### 合成指标文件 (`{SYNTHETIC_ID}.csv`)
```csv
date,value
2020-01-01,0.25
2020-02-01,0.30
...
```

## 📈 数据使用说明

### 1. 数据访问
```python
import pandas as pd

# 读取原始数据
df = pd.read_csv('data/series/T10Y3M.csv', parse_dates=['date'])

# 读取YoY数据
yoy_df = pd.read_csv('data/series/PAYEMS_YOY.csv', parse_dates=['date'])

# 读取合成指标
synthetic_df = pd.read_csv('data/series/CP_MINUS_DTB3.csv', parse_dates=['date'])
```

### 2. 数据验证
```python
# 检查数据完整性
def validate_data(series_id):
    file_path = f'data/series/{series_id}.csv'
    if not os.path.exists(file_path):
        return False
    
    df = pd.read_csv(file_path)
    return not df.empty and 'date' in df.columns and 'value' in df.columns
```

### 3. 数据单位说明
- **百分比**: 利率、利差、比率等，通常以百分比表示
- **指数**: 价格指数、信心指数等，以基准年为100
- **货币单位**: 
  - 百万美元: 企业债务、银行资产等
  - 十亿美元: GDP等大型经济指标
  - 千人: 就业人数等
- **YoY百分比**: 同比增长率，以百分比表示

### 4. 数据频率
- **日**: 每日更新，如利率、汇率等
- **周**: 每周更新，如初请失业金等
- **月**: 每月更新，如就业、通胀等
- **季**: 每季度更新，如GDP等

## 🔄 数据更新流程

1. **数据下载**: 使用`unified_data_pipeline.py`下载最新数据
2. **数据处理**: 自动计算YoY数据和合成指标
3. **数据验证**: 检查数据完整性和格式
4. **数据存储**: 保存到统一的数据目录

## 📝 注意事项

1. **数据延迟**: 部分数据可能存在1-2个月的延迟
2. **数据修正**: FRED数据可能会进行历史修正
3. **数据缺失**: 部分序列在某些时期可能没有数据
4. **单位转换**: 注意不同序列的单位差异
