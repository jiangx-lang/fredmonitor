## 🎯 打分逻辑
- [ ] 组内聚合：加权中位数（没有单项权重就当1）
- [ ] 红线规则：任一核心指标≥80分 ⇒ 该组分≥60
- [ ] 总分红线：若核心≥2个≥80 ⇒ 总分≥40
- [ ] 高风险占比惩罚：>30%指标>70分 ⇒ 总分+5；>50% ⇒ +10
- [ ] 去重/降相关：同组|ρ|>0.7仅保留更前瞻/更稳定的一条

## 📋 基准选择
- [ ] 利差/曲线/压力：对比 crisis_median 或 p25
- [ ] 波动/恐慌：对比 p75
- [ ] 水平类利率：用2000年后样本分位或实质利率
- [ ] 同比向下为险：对比 p25（方向取反）
